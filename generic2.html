Explore app

❮ ❯

Explore Chat application is an application that is made up of Flutter on dart programming language,Node js and Firebase. Explore app is an application with variety of themes of user preferences, various chat functionalities like sending messages from one user to the other, sending images through the user's device's file manager or inbuilt camera functionality for the purpose of sending images from one user to the other. it also have an avenue for uploading pictures on user's profiles page

Explore is an app with full chat functionality which enables a user to connect his/her loved ones around the world with the use of smartphone.
